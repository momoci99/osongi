# 가격 및 수량 변동 추이 차트 강화 계획

> 작성일: 2026-04-19
> 마지막 업데이트: 2026-04-19

## 1. 개요

데이터 분석 페이지의 ChartSection(가격 및 수량 변동 추이)을 개선하여,
사용자가 시리즈 과부하 없이 데이터를 깊이 탐색할 수 있도록 한다.

### 대상 사용자

| 구분               | 환경   | 핵심 니즈                   | 연령대  |
| ------------------ | ------ | --------------------------- | ------- |
| 조합 관계자·분석가 | PC     | 시즌 통계 분석, 보고서 작성 | 40~70대 |
| 채취자·농가        | 모바일 | 내 지역 시세 빠른 확인      | 40~70대 |

### 현재 구현 상태

- ✅ 가격/수량 모드 토글
- ✅ 연도별 서브플롯 (독립 X축, 공유 Y축)
- ✅ 7일/14일 SMA 이동평균선
- ✅ 등급별 색상 + 대시 패턴, 지역별 시리즈
- ✅ 호버 툴팁 (날짜, 지역, 등급, 가격, 수량)
- ✅ PNG 내보내기
- ✅ 반응형 (모바일/데스크톱)
- ✅ 범례 클릭 시리즈 토글 (Phase 0-1)
- ✅ 시리즈 과부하 경고 (Phase 0-2)
- ✅ 차트 확대/축소 — 세로+가로 동시 확장 (Phase 0-3)
- ✅ Crosshair 수직선 + 값 패널 (Phase 1-1)
- ✅ 최고/최저 극값 마커 (Phase 1-2)
- ✅ 연도 오버레이 모드 (Phase 2-1)

## 2. 핵심 문제: 시리즈 과부하

| 선택 시나리오          | 시리즈 수 (연도당)  | 비고                      |
| ---------------------- | ------------------- | ------------------------- |
| 경북 전체 + 전등급     | 13 × 6 = **78개**   | 경북만으로도 과부하       |
| 3개 조합 전체 + 전등급 | 21 × 6 = **126개**  | 범례만 126행              |
| 위 + 5년 기간          | 5 × 126 = **630개** | SVG path + circle 수천 개 |

- `buildYearSeries()`에 시리즈 수 제한·집계 로직 없음
- 필터 UI에서 경고 없이 전체 선택 가능
- 분석 목적 사용자이므로 **선택 자유도는 유지하되, 시각적 관리 도구를 제공**

## 4. 구현 Phase

### Phase 0: 시리즈 과부하 해결 ✅ 완료

#### 0-1. 범례 클릭 시리즈 토글 ✅

범례 항목 클릭으로 해당 시리즈를 show/hide한다.

- **구현**: `useRef` + `data-series-key` 어트리뷰트 방식
  - 시리즈별 `path`/`circle`에 `data-series-key="region-gradeKey"` 부여
  - 범례 클릭 시 ref 업데이트 + `d3.selectAll`로 `opacity`만 토글
  - `useEffect` 의존 배열에 포함하지 않음 → 전체 재그리기 방지
- **온보딩 힌트**: 시리즈 > 20개일 때 범례 위에 "클릭하여 숨기기" 라벨 최초 1회 표시
- **모바일**: 터치 타겟 최소 44px 보장
- **관련 파일**: `useDrawAnalysisChart.ts`, `DataAnalysisChart/index.tsx`

#### 0-2. 시리즈 과부하 경고 ✅

시리즈 수 > 20개일 때 ChartSection 상단에 안내 메시지를 표시한다.

- 메시지: "현재 N개 시리즈가 표시 중입니다. 범례를 클릭하여 관심 시리즈만 남겨보세요"
- 시리즈 수 계산: `countUniqueSeries()` in `seriesBuilder.ts`
- **관련 파일**: `ChartSection.tsx`, `seriesBuilder.ts`

#### 0-3. 차트 확대/축소 ✅

"차트 확대" / "차트 축소" 텍스트 버튼으로 차트를 토글한다.

- `DEFAULT_HEIGHT(400px)` ↔ `EXPANDED_HEIGHT(700px)` 세로 확장
- 확장 시 Grid(7:5) 에서 풀 폭(12) 레이아웃으로 전환 → 가로도 동시 확장
- `expanded` 상태를 `DataAnalysis.tsx`로 끌어올려 Grid 분기 처리
- **관련 파일**: `ChartSection.tsx`, `DataAnalysis.tsx`, `Numbers.ts`

### Phase 1: 핵심 분석 인터랙션 ✅ 완료

#### 1-1. Crosshair (수직선 + 값 패널) ✅

마우스 이동 시 수직선을 표시하고, 해당 날짜의 모든 시리즈 값을 한번에 표시한다.

- **구현**: 서브플롯별 투명 overlay `rect` + `mousemove` 이벤트
- `getNearestDate()`, `formatDateKorean()` 유틸 함수 추가
- D3 Tooltip (`createD3Tooltip`)으로 값 패널 표시
- **관련 파일**: `useDrawAnalysisChart.ts`

#### 1-2. 최고/최저 극값 마커 ✅

각 시리즈의 시즌 내 최고/최저 포인트에 ▲/▼ 마커 + 값 라벨을 자동 표시한다.

- **구현**: `drawExtremeMarkers()` 함수, `showMarkers` 토글 (HeightIcon 버튼)
- 보고서에서 "올해 최고가 언제?" 질문에 즉답
- **관련 파일**: `useDrawAnalysisChart.ts`, `DataAnalysisChart/index.tsx`

---

### Phase 2: 연도 비교 ✅ 완료

#### 2-1. 연도 오버레이 모드 ✅

현재 서브플롯(연도별 좌→우 나열) 외에, 같은 축에 연도별 데이터를 겹쳐 표시하는 모드를 추가한다.

- **구현**:
  - `ChartLayout` 타입 (`"subplot" | "overlay"`), ToggleButtonGroup으로 전환
  - X축: 월-일 기준 (8/1~12/31), `normalizeToBaseYear()` 로 기준년 2000 정규화
  - 색상: 연도별 `YEAR_COLORS` (10색), 대시 패턴: 등급별
  - `buildOverlayYearSeries()`, `OverlaySeries` 타입 추가
  - Crosshair + Tooltip 오버레이 모드 대응
  - 단일 연도 선택 시 overlay 토글 비활성화
  - 레이아웃 전환 시 `hiddenSeriesRef` 자동 초기화
- **관련 파일**: `useDrawAnalysisChart.ts`, `seriesBuilder.ts`, `DataAnalysisChart/index.tsx`

## 5. 기술 제약 사항

| 제약                                                                                          | 영향          | 해결                                         |
| --------------------------------------------------------------------------------------------- | ------------- | -------------------------------------------- |
| `hiddenSeries`를 `useEffect` deps에 넣으면 `svg.selectAll("*").remove()` 호출 → 전체 재그리기 | 0-1 범례 토글 | `useRef` + D3 직접 opacity 조작              |
| 차트→부모 콜백 없음 (시리즈 수 등)                                                            | 0-2 경고      | 부모에서 `chartData`의 unique 조합 직접 계산 |
| `useContainerSize`가 높이 변화에도 반응                                                       | 0-3 높이 확장 | 기능적 정상. 깜빡임 가능하나 수용            |
